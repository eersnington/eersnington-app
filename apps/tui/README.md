# eersnington

This is an Rust TUI version of my site eers.dev which I think is very cool.

```sh
bunx eersnington
```

You can also use `npx` if you don't have Bun installed

```sh
# in great dismay and mortification to Jarred Sumner, proceed to type the following
npx eersnington
```

This CLI is powered by [Ratatui](https://github.com/ratatui/ratatui)

For local development:

```sh
# make mr Sumner happy by downloading bun if you haven't
bun install
bun run dev
```
